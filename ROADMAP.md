# 🗺️ Clone Combat — Roadmap

## v0.1.0 ✅ (Mevcut Sürüm)
- [x] Çok oyunculu temel altyapı (Unity Netcode + Relay + Lobby)
- [x] 2D Platformer oyuncu hareketi ve zıplama
- [x] Silah sistemi (Pistol, Assault, Sniper)
- [x] Ghost (Hayalet) kayıt ve oynatma sistemi
